# Contributing to agentspool Showcase

This showcase repository contains documentation and educational content for the agentspool project.

## For Code Contributions

Code contributions should be made to the main repository:
**[https://github.com/fbratten/agentspool](https://github.com/fbratten/agentspool)**

## For Documentation

If you find errors or want to improve the documentation in this showcase:

1. Open an issue describing the change
2. Fork the repository
3. Make your changes
4. Submit a pull request
